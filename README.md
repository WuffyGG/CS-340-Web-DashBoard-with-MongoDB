# CS 340 Web Dashboard with MongoDB

This repository contains both the original CS 340 artifact and the enhanced CS 499 artifact used in my Computer Science ePortfolio.

## Original Artifact

The original artifact was created in CS 340: Client/Server Development. It consists of a JupyterDash application connected to MongoDB that displays and filters animal shelter records.

## Enhanced Artifact

The enhanced artifact was developed in CS 499 and demonstrates improvements in three areas:

### Software Design and Engineering

- Modular architecture
- Separation of concerns
- Improved maintainability
- Defensive programming

### Algorithms and Data Structures

- Custom rescue suitability scoring algorithm
- Weighted ranking system
- Sorting functionality

### Databases

- MongoDB indexes
- Schema validation
- Aggregation pipelines
- Dashboard summary statistics

## Technologies Used

- Python
- Dash
- MongoDB
- Pandas
- Plotly
- Dash Leaflet
